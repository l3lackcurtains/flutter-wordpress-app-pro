Flutter for WordPress Pro is an extended flutter app of popular open-source flutter for WordPress app ( https://github.com/l3lackcurtains/Flutter-for-Wordpress-App ). If your requirements are minimum then, feel free to use the basic app. The app uses WordPress REST API to fetch the posts from wordpress website. We also provide wordpress plugin which modifies the original wordpress rest API to enhance the performance during the HTTP request. Also, we have used an added cache function to make it more blazing fast.

Overview

Features

  • Beautiful design and blazing fast app
  • Documentation available for set up
  • Latest Posts, Featured Posts, and Categories post in the app
  • Performance-driven from both wordpress and application side
  • Lazy content loading
  • Dark and Light theme
  • Admob integrated
  • Dynamic deep link integrated
  • Supports video in the post thumbnail
  • Supports embedded video inside the post
  • Post Comments and Share
  • One signal notification directly from wordpress app to mobile devices
  • Infinite Scroll
  • Related posts
  • Posts search feature
  • Setting page

Screens

  • Single Post Screen
